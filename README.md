# [OpenClash插件配置文件] 感谢 [【安格视界】](https://github.com/liandu2024/clash) 提供源文件

>> 纯属根据个人使用习惯；专为 OpenClash（基于 Mihomo / Meta 内核）定制的配置文件<br>
>> 在保持丰富分流规则的同时，将路由器处理器和内存开销经量降低<br>
>> 共有两个文件：一个是插件的系统配置文件 运行配置文件<br>
>> ImmortalWrt或者其他OpenWrt 可下载后 使用 `sh xxx.run `来安装<br>
>> 使用 `sh xxx.run --target dir --noexec ` 只解压到dir目录 不执行安装

这是一个专为 OpenClash（基于 Mihomo / Meta 内核）深度定制的终极精简与性能优化配置文件。本配置的核心理念是“极低开销、秒级切换、隐私防漏”，在保持丰富分流规则的同时，将路由器 CPU 和内存开销降到最低
